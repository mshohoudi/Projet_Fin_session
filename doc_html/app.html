app module¶

app.py — Dashboard Streamlit¶

Interface graphique pour le Système Thermoélastique Hybride. Remplace l’interface terminal de main.py.

Lancement :

streamlit run app.py

app.afficher_metriques(res_ana, res_ef, res_ml)¶

Affiche les résultats numériques sous forme de métriques Streamlit.

Les valeurs calculées sont présentées pour les trois méthodes de résolution ainsi que les écarts relatifs entre celles-ci.

Parameters:

  • res_ana (dict) – Résultats du solveur analytique.

  • res_ef (dict) – Résultats du solveur EF.

  • res_ml (dict) – Résultats du modèle ML.

Returns:

Aucun retour.

Return type:

None

app.charger_config()¶

Charge le fichier de configuration YAML.

Le fichier contient les propriétés géométriques, matérielles, numériques ainsi que les paramètres du modèle d’apprentissage.

Returns:

Dictionnaire contenant la configuration du projet.

dict

app.charger_donnees(cfg)¶

Charge les données d’entraînement.

Les données sont importées depuis les simulations Ansys ou générées automatiquement lorsqu’aucun fichier n’est disponible.

Parameters:

cfg (dict) – Dictionnaire de configuration.

Returns:

Jeu de données utilisé pour l’entraînement du modèle.

app.entrainer_modele(_df, cfg)¶

Initialise, valide et entraîne le modèle de substitution.

Une validation croisée K-Fold est réalisée avant l’entraînement final sur l’ensemble des données.

Parameters:

  • _df (pandas.DataFrame) – Jeu de données d’entraînement.

  • cfg (dict) – Dictionnaire de configuration.

Returns:

Modèle de substitution entraîné.

Return type:

ModeleSubstitution

app.fig_comparaison_barres(res_ana, res_ef, res_ml, F, T)¶

Génère les graphiques comparant les résultats des trois méthodes.

Deux diagrammes sont créés :

  • comparaison de la flèche maximale;

  • comparaison de la contrainte maximale de Von Mises.

Parameters:

  • res_ana (dict) – Résultats du solveur analytique.

  • res_ef (dict) – Résultats du solveur EF.

  • res_ml (dict) – Résultats du modèle ML.

  • F (float) – Force appliquée [N].

  • T (float) – Température appliquée [°C].

Returns:

Figure Plotly.

Return type:

plotly.graph_objects.Figure

app.fig_extrapolation_force(modele, cfg, T_fixe)¶

Illustre le comportement du modèle hors de son domaine d’entraînement.

Les prédictions du modèle ML sont comparées à la solution analytique pour une large plage de forces.

Parameters:

  • modele (ModeleSubstitution) – Modèle de substitution entraîné.

  • cfg (dict) – Dictionnaire de configuration.

  • T_fixe (float) – Température maintenue constante [°C].

Returns:

Figure Plotly.

Return type:

plotly.graph_objects.Figure

app.fig_heatmap_3d(df_pred, cible, F, T, label, unite)¶

Génère une cartographie tridimensionnelle des prédictions ML.

Chaque point est coloré selon la valeur de la grandeur sélectionnée.

Parameters:

  • df_pred (pandas.DataFrame) – Résultats prédits sur la grille.

  • cible (str) – Nom de la variable à représenter.

  • F (float) – Force appliquée [N].

  • T (float) – Température appliquée [°C].

  • label (str) – Nom affiché dans la barre de couleurs.

  • unite (str) – Unité de la grandeur.

Returns:

Figure Plotly ou None si la variable est absente.

Return type:

plotly.graph_objects.Figure or None

Génère la grille tridimensionnelle utilisée pour les prédictions ML.

Les coordonnées sont créées à partir de la géométrie de la poutre afin de produire un ensemble régulier de points dans le volume.

Return type:

pandas.DataFrame

app.main()¶

Exécute l’application Streamlit.

Cette fonction coordonne le chargement de la configuration, le chargement des données, l’entraînement du modèle de substitution, l’exécution des solveurs analytique et EF, ainsi que la génération de l’ensemble des graphiques et des tableaux de résultats.

Returns:

Aucun retour.

Return type:

None

app.sidebar(cfg)¶

Construit la barre latérale de l’application Streamlit.

Parameters:

cfg (dict) – Dictionnaire de configuration.

Returns:

Force appliquée [N] et température [°C].

Return type:

tuple[float, float]
